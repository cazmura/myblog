Python は全てがオブジェクトらしい

31 Dec, 2025

Python は全てがオブジェクトらしい。 https://djangostart.com/745/

変数は、箱というより、オブジェクトを特定するためのポインタのようなものらしい。

これを知ったきっかけは、以下のようなコードを見た時に「宣言した変数がなぜメソッドを使えるんだろう？」というところから調べてみたところ、今回の Python の仕様に行き着いた。

lst = [1, 2, 3]
lst.append(4)
print(lst) # [1, 2, 3, 4]
