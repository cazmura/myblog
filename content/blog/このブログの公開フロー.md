このブログは[この記事](https://ingau.me/blog/how-i-write-my-blogs-in-obsidian-and-publish-instantly/)とほぼ同等の仕組みで構築・公開できるようにしている。

ワークフローは以下になる。

1. 対象のディレクトリ直下で  「catnewpost "記事のタイトル"」というコマンドを叩く
2. 生成された記事に本文を記入する
3. ローカルリポジトリにコミットして GitHub にプッシュする
	1. git add .
	2. git commit -m "Publish new post"
	3. git push
