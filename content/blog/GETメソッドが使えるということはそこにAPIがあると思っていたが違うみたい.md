GETメソッドとAPIは別の概念です。

**GETメソッド：**

- HTTPプロトコルの基本的なメソッドの一つ
- すべてのWebサーバーが対応している標準機能
- 普通のWebページ（HTML）を取得するときも使われています

**API：**

- アプリケーション間でデータをやり取りするための仕組み
- 通常はJSON/XMLなどの構造化データを返す
- 意図的に設計・実装されたもの

**例で説明すると：**

bash

```bash
# 普通のWebページ（APIではない）
GET /index.html → HTMLが返る

# Web API
GET /api/users/123 → JSONデータが返る
```

**info.cern.chの例：** あなたが先ほど試したのは、世界初のWebサイトの普通のHTMLページを取得しただけで、APIではありません。ブラウザで見るのと同じものを、telnetで手動取得しただけです。

**まとめ：**

- GETメソッド = HTTPの基本機能（すべてのWebサーバーにある）
- API = 特別に設計されたデータ提供の仕組み（明示的に実装が必要）

GETメソッドはAPIでもよく使われますが、単なるWebページの取得でも使われるということです。
